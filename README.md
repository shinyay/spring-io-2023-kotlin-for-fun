## Description

A simple HTTP application.
Data operation is performed via REST using Spring Data JPA.

## Demo

Run App:

```shell
$ ./gradlew clean bootRun
```

End point access:

```shell
$ curl -X GET localhost:8080/api/v1/books
```

## Requirement

My environment:
- JDK: 19
- OS: MacOS
